# student-finance-advisor

A smart financial assistant designed to help students track expenses and get AI-driven savings advice.

###  Features
- **Expense Tracking:** Add and view daily expenses easily.
- **AI Insights:** Powered by Google Gemini to identify overspending patterns.
- **Budgeting:** Set monthly targets and track progress.

### Tech Stack
- **Language:** Python
- **UI:** Streamlit
- **AI:** Google Gemini API

### Setup
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/student-finance-advisor.git`
2. Install requirements: `pip install -r requirements.txt`
3. Add your `GEMINI_API_KEY` to `.streamlit/secrets.toml`.
4. Run the app: `streamlit run app.py`
